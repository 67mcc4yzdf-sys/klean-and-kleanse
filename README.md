# FreshNest Cleaning Template

A reusable, conversion-focused Next.js website for local cleaning companies.

## Customize

Edit `lib/business.ts` to change the business name, city, contact details, services, colors, reviews, FAQs, hours, images, social links, and service areas. Replace the generated preview images in `public/images` or update their paths in the same config.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
